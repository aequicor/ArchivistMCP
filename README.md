# ArchivistMCP

MCP-server на Kotlin для индексации документов с ChromaDB.

## Технологии

| Компонент | Версия |
|-----------|--------|
| Kotlin | 2.2.21 |
| Ktor | 3.3.3 |
| MCP Kotlin SDK | 0.11.1 |

## Транспорт

- **STDIO** — для локальных агентов
- **SSE** — для удалённого подключения (порт 8080)

## Инструменты

- `index_document` — индексация документа
- `query_documents` — семантический поиск
